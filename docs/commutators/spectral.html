Spectral curves of commuting linear differential operators¶

Computation of spectral curves.

This module contains the main functionality to compute and analyze the spectral curves obtained from a pair of commuting linear differential operators.

Elements provided by the module¶

dalgebra.commutators.spectral.spectral_operators(L, P, name_lambda='lambda_', name_mu='mu')¶

Method to create the spectral operators associated with two differential operators.

This method assumes \(L\) and \(P\) are two linear differential operators in the same parent of the form \(F[\textbf{x}]\{z\}\), i.e., one differential variable, then a set of algebraic variables (may or not be constant) and a base field \(F\) such that \(\partial(F) = 0\).

To create the spectral operators we need to add algebraically the two constants \(\lambda\) and \(\mu\) to \(F[\textbf{x}]\) at the same level. Then we will need to add again the variable \(z\).

This method will then return the two operators \(L_\lambda = L - \lambda\) and \(P_\mu = P - \mu\).

dalgebra.commutators.spectral.SpectralCurveOverIdeal(loglevel, logfile, *args, **kwds)¶

Method that automatizes the computation of spectral curve and some extra data throughout the solution branches of an ideal.

dalgebra.commutators.spectral.BC_pair(L, P)¶

Algorithm BC_pair from doi:10.3842/sigma.2019.101 (Section 6).

This method takes as input two operators that commute and computes a pair \((L,B)\) that is a BC pair and its order.
